# Isolation Forest

## Description
A simple Isolation Forest Model.

## Inputs
* **n_estimators** (int): The number of base estimators in the ensemble.
* **train_data** (Pandas DataFrame): The training data set.
* **test_data** (Pandas DataFrame): A test data set on which the trained model is applied immediately. 

## Outputs
* **trained_model** (sklearn Isolation Forest instance): The trained model. This object can be stored and loaded in another workflow.
* **decision_function_values** (Pandas Series): The decision function values of the model applied to the test data.

## Details
Isolation Forest from sklearn.
